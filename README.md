# Cozy Coder

Tools for your AI coding agent.

## Prerequisites

- [mise](https://mise.jdx.dev)

## Usage

Setup local environment:

```bash
mise install
mix ecto.create
mix setup
```

Run daemon in development mode:

```bash
mix phx.server
```

Launch [MCP inspector](https://modelcontextprotocol.io/docs/tools/inspector):

```bash
mise run mcp
```

Run tests:

```bash
mix test
```
